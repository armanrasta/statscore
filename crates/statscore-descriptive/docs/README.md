# statscore-descriptive

Descriptive statistics: univariate summaries, bivariate measures, rank correlations, and robust estimators.

## Planned modules

| Module | Contents |
|--------|----------|
| `univariate` | Mean, variance, std, skewness, kurtosis, quantiles |
| `bivariate` | Covariance, Pearson r |
| `rank` | Spearman ρ, Kendall τ |
| `robust` | Median, IQR, MAD, trimmed mean |
| `grouped` | By-group statistics (rayon parallel) |

## Python bindings

Exposed as `statscore.descriptive` with NumPy array inputs.

## Dependencies

- `statscore-common`
