# statscore-multivariate

Multivariate analysis: PCA, clustering, classification, and MANOVA.

## Planned modules

| Module | Contents |
|--------|----------|
| `reduction` | PCA (SVD), factor analysis, classical MDS |
| `clustering` | K-means, hierarchical, DBSCAN |
| `classification` | LDA, QDA |
| `manova` | Multivariate analysis of variance |

## Dependencies

- `statscore-common`, `statscore-linalg`, `statscore-distributions`

## Python bindings

`statscore.multivariate` — PCA `fit_transform()` returning NumPy arrays.
