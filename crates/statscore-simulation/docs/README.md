# statscore-simulation

Simulation methods: bootstrap, permutation tests, Monte Carlo, and power simulation.

## Planned modules

| Module | Contents |
|--------|----------|
| `bootstrap` | Nonparametric, parametric, percentile and BCa CIs |
| `permutation` | Permutation tests |
| `monte_carlo` | Generic Monte Carlo integration |
| `quasi_monte_carlo` | Sobol sequences |
| `power_simulation` | Simulation-based power analysis |

## Dependencies

- `statscore-common`, `statscore-distributions`

## Python bindings

`statscore.simulation` — `bootstrap(data, statistic, n_resamples)` API.
