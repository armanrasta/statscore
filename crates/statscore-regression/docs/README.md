# statscore-regression

Regression models: OLS, GLM, and regularized regression with diagnostics.

## Planned modules

| Module | Contents |
|--------|----------|
| `linear` | OLS (QR), WLS, GLS |
| `glm` | Logit/probit links, Binomial/Poisson/Gaussian, IRLS |
| `regularized` | Ridge, Lasso, Elastic Net |
| `diagnostics` | Residuals, leverage, Cook's D, VIF |

## Dependencies

- `statscore-common`, `statscore-linalg`

## Python bindings

`statscore.regression` — `fit()` / `predict()` API similar to statsmodels/sklearn.
