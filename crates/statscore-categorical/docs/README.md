# statscore-categorical

Categorical data analysis: contingency tables, agreement, and log-linear models.

## Planned modules

| Module | Contents |
|--------|----------|
| `contingency` | χ², Fisher exact, odds ratio |
| `agreement` | Cohen's κ, Fleiss κ |
| `loglinear` | Log-linear models |

## Dependencies

- `statscore-common`, `statscore-hypothesis`

## Python bindings

`statscore.categorical` — crosstab and κ statistics.
