# statscore-timeseries

Time series analysis: stationarity, smoothing, ARIMA, and decomposition.

## Planned modules

| Module | Contents |
|--------|----------|
| `stationarity` | ADF, KPSS |
| `smoothing` | EMA, Holt, Holt–Winters |
| `arima` | AR, MA, ARMA, ARIMA |
| `decomposition` | Trend + seasonal + residual |

## Dependencies

- `statscore-common`, `statscore-linalg`

## Python bindings

`statscore.timeseries` — `fit()` / `forecast()` API.
