# statscore-information

Information theory: entropy, divergences, and mutual information.

## Planned modules

| Module | Contents |
|--------|----------|
| `entropy` | Shannon, Rényi, Tsallis, differential |
| `divergence` | KL, JS, Hellinger, total variation |
| `mutual_info` | MI, conditional MI, transfer entropy |

## Dependencies

- `statscore-common`, `statscore-special`

## Python bindings

`statscore.information` — histogram-based MI with NumPy inputs.
