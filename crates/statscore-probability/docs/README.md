# statscore-probability

Probability theory primitives: moments, inequalities, and distribution transforms.

## Overview

Bridges `statscore-special` (special functions) and `statscore-distributions` (concrete distributions). Provides reusable probability identities and transforms.

## Planned modules

| Module | Contents |
|--------|----------|
| `moments` | Raw/central moments, MGF, characteristic functions |
| `inequalities` | Chebyshev, Markov, Hoeffding, Jensen |
| `transforms` | PIT, copula helpers, convolutions |

## Dependencies

- `statscore-common`
- `statscore-special`
- `statscore-linalg`

## Status

**Scaffold** (Phase 0). Implementation not started.
